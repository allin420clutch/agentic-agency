## SECTION 2: THE 5-STEP RESEARCH PROCESS FLOW DIAGRAM

**Flow Structure:** Linear, left-to-right progression with 5 connected steps

**Step 1 — Find the Community**  
→ Icon: People/group icon  
→ Description: Locate where your audience gathers online (Facebook groups, subreddits, forums)  
→ Key Question: "Where do they hang out?"

**Step 2 — Learn Their Language**  
→ Icon: Speech bubble / quote icon  
→ Description: Note the phrases, inside jokes, and terminology they use naturally  
→ Key Question: "What words do they use to describe themselves?"

**Step 3 — Validate Interest**  
→ Icon: Chart trending up  
→ Description: Use Google Trends, Etsy bestsellers, and Pinterest to confirm demand exists  
→ Key Question: "Is search interest steady or rising?"

**Step 4 — Run the 3-Question Test**  
→ Icon: Checklist / test icon  
→ Description: Confirm real community, name 5 products, identify phrase potential  
→ Key Question: "Does this niche pass all three?"

**Step 5 — Design With Direction**  
→ Icon: Pencil/design tool icon  
→ Description: Create products that speak directly to what you discovered  
→ Key Question: "Am I designing to fill demand?"

**Visual Style:**  
- Connected nodes with arrows between each step  
- Each step in a rounded card/circle  
- Use alternating accent colors (coral, mint, coral, mint, coral)  
- Add small checkmark icons in corners to show progression

## SECTION 5: RED FLAGS vs. GREEN LIGHTS COMPARISON

**RED FLAGS (Step Back)**

| Warning | What It Means |
|---------|---------------|
| Every design looks identical | No room to differentiate — same clipart recycled |
| Prices at $8-10 across the board | Margin destroyed — race to the bottom |
| One style dominates completely | No unique angle possible |
| Audience can't explain their identity | No emotional hook for designs |
| Only passing trend interest | Viral spike with no sustained demand |
| No searchable communities | No target audience to reach |

**GREEN LIGHTS (Proceed)**

| Signal | What It Means |
|--------|---------------|
| Inside jokes & shared language exist | Strong identity-expression potential |
| Products are emotional, not just decorative | Makes people feel seen = easier sales |
| 3+ active communities found | Clear audience to target |
| Predictable seasonal spikes | Can plan inventory around demand |
| Community has special titles/names | Built-in phrases for designs |
| Rising or steady trend line | Sustainable, ongoing interest |

**Visual Direction:**  
- Split layout: Red column on left with coral/red background, Green column on right with mint/green background  
- Use X icons for red flags, checkmark icons for green lights  
- Make this highly scannable — bold keywords, short lines

## DESIGN NOTES FOR COMPLETE INFOGRAPHIC

**Recommended Dimensions:** 1200px × 2400px (vertical scroll format) or 1920px × 1080px (presentation/slide format)

**Color Palette Summary:**
- Background: #1A2744 (deep navy)
- Primary accent: #FF6B6B (coral red)
- Secondary accent: #4ECDC4 (mint teal)
- Text primary: #FFFFFF
- Text secondary: #B8C5D6

- Green light tint: #1F3D2D

**Typography:**
- Headlines: Bold sans-serif, all caps, large (32-48pt)
- Subheadlines: Medium weight, smaller (18-24pt)
- Body: Regular weight, high contrast (14-16pt)
- Stats callouts: Bold, extra large numbers, accent colors

**Visual Elements:**
- Rounded corners on all cards/boxes (8-12px radius)
- Subtle drop shadows for depth
- Icons for each step in the flow diagram (simple, line-art style)
- Progress indicators showing the 5-step sequence
